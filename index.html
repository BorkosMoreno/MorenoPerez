Cómo publicarla gratis

Sigue estos pasos para tenerla online sin coste, con tu dominio de IONOS.

  1. 1

    Crea un repositorio en GitHub

    Entra en GitHub y crea un repositorio nuevo (público). Puedes llamarlo, por ejemplo, mi-web.

  2. 2

    Sube el archivo index.html

    Dentro del repositorio pulsa “Add file → Upload files” y arrastra este archivo index.html. Luego pulsa Commit changes.

  3. 3

    Activa GitHub Pages

    Ve a Settings → Pages. En Branch elige main y carpeta / (root), y guarda. En unos minutos tu web estará en https://tuusuario.github.io/mi-web.

  4. 4

    Conecta tu dominio de IONOS

    En GitHub, en Settings → Pages → Custom domain, escribe tu dominio (ej. tudominio.com) y guarda. Esto crea un archivo CNAME.

    Luego, en el panel de IONOS → Dominios → DNS, configura:

    Registros A (para tudominio.com) apuntando a:
    185.199.108.153
    185.199.109.153
    185.199.110.153
    185.199.111.153
    Registro CNAME (para www) apuntando a:
    tuusuario.github.io
  5. 5

    Espera y activa HTTPS

    Los cambios de DNS pueden tardar de minutos a 24 h. Cuando esté listo, marca en GitHub Pages la casilla “Enforce HTTPS”. ¡Listo, tu web estará online y con candado seguro!
